# 문제

- 클래스의 상속을 코드로 구현해보시오.
- 이 문제는 자바 OOP 문법에서 상속에 대한 개념과 그것을 코드로 구현할 수 있는지를 묻는 문제이다.

- 이름 : 원더우먼, 나이 : 30, 성별 : 1, 파워 : 100
